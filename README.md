Whether you're starting your DevOps journey or improving your Linux automation skills, these scripts provide hands-on practice with common scripting tasks.

```bash

```

## 🚀 Getting Started

### Clone the Repository

```bash
git clone https://github.com/your-username/Shell-Scripting.git
```

```bash
cd Shell-Scripting
```

---

## ▶️ Make Scripts Executable

Grant execute permission to all scripts:

```bash
chmod +x *.sh
```

Or for a single script:

```bash
chmod +x Script1.sh
```

---

## ▶️ Run a Script

Using Bash:

```bash
bash Script1.sh
```

Or directly:

```bash
./Script1.sh
```
